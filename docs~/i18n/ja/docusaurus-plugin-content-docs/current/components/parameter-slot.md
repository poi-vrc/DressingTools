# パラメータスロット

*2.5.0 以降*

パラメータスロットコンポーネントは、複数のSmartControlコンポーネント間で共有されるタグとして機能します。
これらの SmartControls には特定の値が割り当てられ、1つの `int` または `float` アニメーターパラメーターによって制御されます。
パラメータ値が一致するSmartControlだけが有効になります。

この仕組みはUnity animatorのAny Stateに似ていますが、内部的には使われていません。

![Group Dynamics](/img/comp-parameter-slot.png)

## マッピング

マッピングは、パラメータスロット コンポーネントのインスペクタまたは SmartControl ドライバのインスペクタで
直接編集できます。SmartControl のいずれかがデフォルト値でマッピングされている必要があります。

## 設定

- **パラメータ名**: `int`/`float` パラメータの名前。空白の場合は自動的に生成されます
- **値タイプ**: 値の型は `Int` または `Float` であります
- **デフォルト値**: パラメータデフォルト値
- **ネットワーク同期**: このパラメータをネットワーク同期させるかどうか
- **保存される**: このパラメータをワールド間で保存するかどうか
