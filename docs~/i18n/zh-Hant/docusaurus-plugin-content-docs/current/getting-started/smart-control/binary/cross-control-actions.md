# 交叉控制操作

交叉控制動作允許智能控制同時控制另一個智能控制。

目前只開發了一種交叉控制動作。 未來將添加更多。

## 值

分別在啟用和禁用時直接將另一個智能控制設定為特定值。

![Cross-control value](/img/smartcontrol-crossctrl-value.PNG)

**用例範例：**
- 換衣服只顯示一件衣服
   - 當此智能控制啟用時關閉其他智能控制。
- 透過啟用其他服裝的配件來混合服裝
   - 當此智能控制啟用時，開啟其他智能控制。

:::info
對於二進位智能控制/布林參數，值 `0` 表示 `false`，`1` 表示 `true`。
:::
